# Carpeta core

Contiene lógica/métodos/servicios/tipados/modelos que pueden ser reutilizados dentro de la aplicación.
No contiene nada relacionado a componentes visuales.

Incluye:
- Modelos
- DTOs
- Servicios HTTP
- Guards

No debe contener:
- Componentes UI
- Pipes visuales
- Elementos de presentación
- Features
